# Gurgaon-real-estate-analysis
Gurgaon Real Estate Data Analysis using Python (EDA project) to explore pricing trends based on location, area, BHK, property type, and builder. Insights extracted using Pandas, Seaborn, and Matplotlib.
# 🏡 Gurgaon Real Estate Data Analysis

## 📌 Project Overview
This project explores Gurgaon real estate data using Python to understand how different factors like location, area, BHK, builder, and property type influence property prices.

The goal was to perform **Exploratory Data Analysis (EDA)** and extract meaningful insights from real-world housing data.

---

## 🎯 Key Objectives
- Identify the most expensive properties and locations  
- Understand how price varies with area and BHK  
- Compare ready-to-move vs under-construction properties  
- Analyze price per square foot across localities  
- Study the impact of builders and property types on pricing  

---

## 🔍 Key Insights

- 💰 The costliest property in the dataset is priced at ₹12,263 crore in  DLF Camellias
- 📍 Baliawas has the highest average property price
- 📏 Sector 113 records the highest price per sq ft at ₹58,628/sqft
- 🏗️ Ready-to-move properties are generally more expensive than under-construction ones
- 🏢 RERA-approved properties show a noticeable price premium
- 📐 Strong positive correlation between area and total price
- 🛏️ 4 BHK properties are the most expensive on average
- 🏘️ Certain property types Residential Plots dominate in pricing
- 🏗️ Some builders consistently price their properties higher than others
- 📉 Larger homes tend to have lower price per square foot (economies of scale effect)
